### Final_project.ipynb
Оценка retention и A/B-тестирование акционных предложений: 
- определение и оценка retention rate игроков на основе специально подготовленной функции, в которую надо указать путь к файлу (Python - Pandas, Seaborn);
- проведение A/B тестирования для определения наборов акционных предложений для игроков на основе revenue, arpu, arppu, CR (статистический анализ и бутстрапирование данных);
- оценка и предложение необходимых метрик для удержания игроков.

### cohort_retention_analysis.ipynb
Когортный анализ и расчёт Retention (Python + pandas)
Задача: определить, как меняется удержание пользователей в зависимости от месяца их первого визита (когорты).
Что делает скрипт:
- Генерирует демо-данные о действиях пользователей
- Строит когорты по месяцу регистрации
- Рассчитывает Retention Rate (удержание) для каждого месяца жизни когорты
- Визуализирует тепловой картой
Используемые библиотеки: pandas, numpy, matplotlib, seaborn

### sql_window_functions.sql
Задача:
Для каждого пользователя посчитать:

- общее число действий,
- ранжирование пользователей по активности в рамках каждого месяца,
